AI Bot Pro

AI Bot Pro is an AI-powered interactive website assistant designed to enhance user engagement and navigation. It can auto-scroll to sections, provide intelligent page briefings, and respond to user commands using both speech and text.

Features

-Smooth auto-scroll to website sections based on user commands
-Intelligent page briefings with speech and text output
-Integration with Google Generative AI API for natural conversation
-Simple and responsive frontend interface
-Backend built with Node.js and Express

Usage

After cloning your repo, users can simply run:

npm install
npm start


And if developing:

npm run dev

Technologies Used

Node.js
Express.js
HTML, CSS, JavaScript
Google Generative AI API
SpeechSynthesis API (for voice output)

Usage

Type or speak commands like:

"Go to about page"
"Show contact section"
"Tell me about this page"
The assistant will automatically scroll to the requested section and describe it.
